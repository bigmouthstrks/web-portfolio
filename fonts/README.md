# Half-Life 2 Font Setup

## Cómo obtener la fuente Half-Life 2

Para usar la tipografía oficial de Half-Life 2 en este portafolio, necesitas descargar el archivo `halflife2.ttf` desde:

**🔗 [DaFont - Half-Life 2 Font](https://www.dafont.com/halflife2.font)**

### Pasos para la instalación:

1. **Descargar la fuente:**

   - Ve a https://www.dafont.com/halflife2.font
   - Haz clic en "Download" para descargar `halflife2.ttf`

2. **Colocar el archivo:**

   - Coloca el archivo `halflife2.ttf` en la carpeta `fonts/`
   - La estructura final debe ser:

   ```
   fonts/
   ├── halflife2.ttf
   ├── halflife2.css
   └── README.md
   ```

3. **Verificar la instalación:**
   - Abre el portafolio en el navegador
   - Los números que aparecen solos (como "4+", "C2", porcentajes) deberían usar la tipografía Half-Life 2

## Elementos que usan la fuente Half-Life 2

La tipografía se aplica específicamente a:

- **Números de estadísticas** (4+, C2, etc.)
- **Porcentajes de habilidades** (95%, 92%, etc.)
- **Elementos del HUD** (si están habilitados)
- **Contadores de volumen**

## Fallback

Si la fuente no está disponible, el sistema usará automáticamente:

1. `HalfLife2Fallback` (Orbitron)
2. `Orbitron` (fuente similar)
3. `monospace` (fuente del sistema)
